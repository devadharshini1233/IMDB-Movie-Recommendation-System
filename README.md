# 🎬 IMDb Movie Recommendation System Using Storylines

## 📌 Overview
This project is a Content-Based Movie Recommendation System that suggests movies based on storyline similarity using Natural Language Processing (NLP).

---

## 🚀 Features
- Recommend movies based on selected movie
- Recommend movies based on user-entered storyline
- Uses TF-IDF Vectorization
- Uses Cosine Similarity
- Interactive UI using Streamlit

---

## 🧠 Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- NLP (TF-IDF)
- Cosine Similarity

---

## 📊 Dataset
- TMDB 5000 Movies Dataset
- Contains movie titles and storylines

---

## ⚙️ How It Works
1. Movie storylines are cleaned and processed
2. TF-IDF converts text into numerical vectors
3. Cosine similarity is calculated between movies
4. Top 5 similar movies are recommended
